Netflix Clone 🎬

A responsive Netflix-style landing page built with HTML, CSS, and JavaScript. Features include a hero section, streaming showcase, FAQ accordion, login/signup authentication using localStorage, and email validation.

🚀 Live Demo

(You can host this on GitHub Pages, Vercel, or Netlify)

Features
Hero section with email input and call-to-action
Showcase of TV, mobile, and device streaming
Video overlays for realistic Netflix preview
FAQ section with expandable/collapsible answers
Login and Sign Up popup authentication using localStorage
Email validation and error handling
Logout functionality
Responsive design for desktops, tablets, and mobile devices
Modern Netflix-like UI and styling
Technologies Used
HTML5 – Structure and content of the website
CSS3 – Styling, layout, and responsive design
JavaScript (ES6+) – FAQ toggle, login/signup functionality, email validation, localStorage management

How to Use
Clone the repository or download the project.
Open index.html in a web browser.
Use the Login / Sign Up buttons to create an account or log in.
Enter your email in the hero section and click Get Started to see the simulated Netflix flow.
Explore FAQ by clicking questions to expand/collapse answers.
Logout using the Logout button after login.
